HTML Structure

An HTML document is always built up from the following elements:

Tag Name Description
<!DOCTYPE html> document type declaration A declaration to the browser about what type of document to expect. In this case: HTML.
<html> html element The root of the html document. All content should be placed between its opening an closing tag.
<head> head element Contains the metadata of the html document. E.g. the page title and links to CSS documents.
<body> body element Represents the concent of the html document. Content of the body is rendered by the browser.

HTML Tables

This section will describe how to make HTML tables.

Table elements

Tag Name Description
<table> table element The wrapper element for all HTML tables.
<thead> table head element The set of rows defining the column headers in a table.
<tbody> table body element The set of rows containing actual table data.
<tfoot> table footer element The set of rows defining the footer in a table.
<th> table heading element Defines cells as headers of columns or rows.
<tr> table row element The table row container.
<td> table data element The table cell container.

Table attributes

Attribute Name Description
colspan colspan attribute Defines how many columns a td element should span.
rowspan rowspan attribute Defines how many rows a td element should span.
